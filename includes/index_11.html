Felder - Stand Bisher

  • Wir nutzen elementare Datentypen, in denen wir einen Wert speichern können.
  • Jetzt wollen wir viele Daten speichern.
  • Dafür müssten wir für jedes Datum eine eigene Variable erstellen.
    • int zahl1 = 10;
int zahl2 = 15;
int zahl3 = 20;
int zahl4 = 25;
int zahl5 = 30;
  • Das ist uns zu viel Aufwand!

Felder

  • Viel einfacher geht dies mit Feldern.
  • In einem Feld werden sozusagen mehrere Variablen gespeichert.

Vorstellung

Im Code



int[] feld = new int[5];
feld[0] = 10;
feld[1] = 15;
feld[2] = 20;
feld[3] = 25;
feld[4] = 30;

Felder

  • Die Initialisierung kann man noch verkürzen.
  • Bei vorher bekannten Werten kann man das Feld auch direkt Initialisieren.
    • int[] feld = {10, 15, 20, 25, 30};
  • Die größe wird durch die Anzahl der Zahlen in den geschweifen Klammern festgelegt.

Felder - Mehrdimensional

  • Manchmal benötigt man mehr als eine Dimension (ein Feld von Feldern).
  • Mit weiteren [] lässt sich die Dimension von Feldern erhöhen.

Vorstellung

Im Code



int[][] feld = new int[3][2];
feld[0][0] = 10;
feld[0][1] = 15;
feld[1][0] = 20;
feld[1][1] = 25;
feld[2][0] = 30;
feld[2][1] = 35;
10 15
20 25
30 35

Felder - Mehrdimensional

  • Mehrdimensionale Felder können auch direkt Initialisiert werden.
  • Dann ist es auch möglich die größen der Felder in den unteren Dimensionen zu Varieren.

Vorstellung

Im Code



int[][] feld = {{10,15},{20,25,30},{35}};
10 15
20 25 30
35

Aufgabe 32 - "Felder IV"

  • Fragt nach einem Integer Wert und lest diesen ein.
  • Erzeugt ein eindimensionales Feld der Länge des eingelesenen Wertes.
  • Initialisiert das Feld mit dem Index der Position.
    • feld[i] = i;
  • Gebt das Feld auf der Konsole aus.
  • Hinweis: mit .length kann die Länge eines Feldes abgefragt werden.
    • feld.length

Aufgabe 32 - "Felder IV"
(Lösung)

import java.util.Scanner;
class Aufgabe32 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Bitte gebe die Länge des Feldes ein!");
        int länge = scanner.nextInt();
        int[] feld = new int[länge];
        for(int i = 0; i < feld.length; i++) {
            feld[i] = i;
        }
        for(int i = 0; i < feld.length; i++) {
            System.out.println(feld[i]);
        }
    }
}

Aufgabe 33 - "Felder V"

  • Programmiere eine Namensliste.
  • Frage zuerst wie viele Namen eingespeichert werden sollen.
  • Dann erzeuge ein String Feld mit entsprechender Länge.
  • Frage nach den Namen, um das Feld zu Initialisieren.
  • Frage in einer Dauerschleife nach einem Index, welcher Ausgegeben werden soll und gebe den eintragt an dieser Stelle aus.

  • Hinweis: Eine Dauerschleife ist eine while-Schleife mit true als Bedingung.

Aufgabe 33 - "Felder V"
(Lösung)

import java.util.Scanner;
class Aufgabe33 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Bitte gebe die Anzahl der Namen ein!");
        int länge = scanner.nextInt();
        String[] name = new String[länge];
        for(int i = 0; i < name.length; i++) {
            System.out.println("Name :");
            name[i] = scanner.next();
        }
        while(true) {
            System.out.println("Welchen Eintrag möchten Sie sehen?");
            int eintrag = scanner.nextInt();
            System.out.println("Name: " + name[eintrag]);
        }
    }
}

Aufgabe 34 - "Felder VI"

  • Erweitere dein Programm aus Aufgabe 33 zu einer Telefonliste.
  • Frage zusätzlich zum Namen auch nach zwei Telefonnummern.
  • Gebe diese zusätzlich zum Namen aus.

Aufgabe 34 - "Felder VI"
(Lösung)

import java.util.Scanner;
class Aufgabe33 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Bitte gebe die Anzahl der Namen ein!");
        int länge = scanner.nextInt();
        String[] name = new String[länge];
        int[][] telefonnummer = new int[länge][2];
        for(int i = 0; i < name.length; i++) {
            System.out.println("Name :");
            name[i] = scanner.next();
            System.out.println("Telefonnummer1 :");
            telefonnummer[i][0] = scanner.nextInt();
            System.out.println("Telefonnummer2 :");
            telefonnummer[i][1] = scanner.nextInt();
        }
        while(true) {
            System.out.println("Welchen Eintrag möchten Sie sehen?");
            int eintrag = scanner.nextInt();
            System.out.println("Name: " + name[eintrag] 
            + " Telefonnummern: " + telefonnummer[eintrag][0] + ", " + telefonnummer[eintrag][1]);
        }
    }
}
